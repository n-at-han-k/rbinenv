# rbinenv

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/n-at-han-k/rbinenv)

A template for creating Ruby gems.

## Usage

```bash
git clone https://github.com/n-at-han-k/rbinenv
cd rbinenv
bin/rename-gem my-new-gem
bin/update-spec
bin/choose-license
bin/increment-version patch
bin/release-gem
```
